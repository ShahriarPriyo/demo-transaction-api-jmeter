# demo-transaction-api-jmeter
API testing using jmeter for a demo e-commerce site's API

## Pre-requisite (Used versions for this project)
- Java STL(version 11.0.16.1 used here)
- Java(TM) SE Runtime Environment 18.9
- Apache Jmeter 5.6.2
